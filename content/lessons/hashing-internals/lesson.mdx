<Intuition>
A hash map is a bet: sacrifice all ordering, and in exchange answer "have I
seen this key?" in constant time. The bet is honest only while keys spread
evenly across buckets. The visual below lets you toggle a bad hash function
and watch the structure quietly renege — chains grow, and O(1) lookup decays
into a linear scan. Every database index, DNS cache, and deduped event
stream is this diagram wearing a production budget.
</Intuition>
